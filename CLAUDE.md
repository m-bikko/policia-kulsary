@AGENTS.md

## Проект

Jylyoi Police - trilingual (kz/ru/en) taplink-сайт отдела полиции Жылыойского района (г. Кульсары, Атырауская область, МВД РК). Next.js 16 App Router + Tailwind v4 + motion. `pnpm dev` / `pnpm build`.

## Wiki (База знаний)

В проекте есть LLM-поддерживаемая база знаний в `wiki/`.

**При любых вопросах о проекте:**
1. Сначала прочитай `wiki/index.md` - найди релевантные страницы.
2. Прочитай нужные страницы из `wiki/pages/`.
3. Отвечай, опираясь на вики. Если информации нет - ищи в коде и обновляй вики.

**При изменениях в проекте:**
- Если изменение затрагивает архитектуру, i18n, дизайн-систему или контент-факты - обнови соответствующие страницы в `wiki/pages/`.
- Обнови `wiki/index.md`, если создана новая страница.
- Добавь запись в `wiki/log.md` (дата, тип операции, затронутые страницы).

**Схема вики:** `wiki/CLAUDE.md`

## Ключевые правила

- Любой UI-текст - сразу во все три словаря `lib/i18n/dictionaries/` (тип `Dictionary` форсирует).
- Дизайн - только через токены из `app/globals.css` (navy+gold), карточки - `.card-official`.
- Иконки - lucide-react/инлайн SVG, эмодзи в UI запрещены.
- Тач-таргеты ≥44px, анимации только transform/opacity + reduced-motion.
